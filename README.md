# 🚀 天平數據自動化輸入系統 (Weight-to-Excel)

本系統專為 Mettler Toledo 天平設計，能將天平輸出的數值自動、精準地填入 Excel 儲存格中。透過 **Excel COM API** 技術，解決了傳統模擬鍵盤輸入容易受到輸入法干擾、視窗焦點必須在最前面的痛點。

---

## 🌟 核心特色

1. **直接寫入技術**：資料直接導入 Excel 活頁簿，即使您正在使用其他程式（如瀏覽網頁或查資料），天平數據依然能正確填入 Excel 背景中的儲存格。
2. **不干擾輸入法**：完全不受中文/英文輸入法的切換影響，解決「重量變成注音」的問題。
3. **雜訊過濾**：自動排除天平選單設定中的雜訊文字（例如：Time, Standby 等），僅抓取真正的重量數值。
4. **免 VBA 部署**：不需在每個活頁簿中貼入程式碼。只要一個 `.exe`，即可適用於所有 Excel 檔案。
5. **互動式設定精靈**：第一次啟動時會引導設定 COM Port、通訊參數與游標方向，並儲存於 local 設定檔中。
6. **安全斷開機制**：
    * **主動斷開**：關閉視窗即立即釋放 COM Port，方便使用者切換回 LIMS 系統。
    * **閒置防佔用**：若超過 30 分鐘(可自訂)無數據，系統會自動斷開，避免資源長期佔用。

---

## 📁 專案檔案結構

* `WeightToExcel.exe`：主要執行檔（由 `.ps1` 編譯而來）。
* `WeightToExcel.ps1`：原始程式碼。
* `config.json`：儲存您天平與通訊設定的設定檔（第一次啟動後產生）。

## 🛠️ 使用說明

### 1. 環境準備

* 將天平透過 RS232 或轉 USB 線連接至電腦。
* 開啟您要記錄重量的 Excel 活頁簿。

### 2. 第一次啟動 (或重新設定)

1. 雙擊執行 `WeightToExcel.exe`。
2. 若為第一次啟動，系統會列出偵測到的 COM Port，並請您確認：
    * **COM Port**：選擇連接天平的端口（例如 COM1）。
    * **下一格游標移動方向**：輸入完畢後要「往下跳 (down)」或「往右跳 (right)」。
    * 其餘參數（Baud Rate 等）若無特殊需求，直接按 **Enter** 使用預設值即可。

### 3. 日常稱重流程

1. 開啟 `WeightToExcel.exe`。
2. 在 Excel 中選定儲存格。
3. 在天平穩定後按下 **Print** 鍵，數值即填入 Excel 並自動跳往下一格。
4. 稱重結束後，直接**關閉**黑色監控視窗，系統會釋放 Excel 控制權，方能切換回純文字模式傳輸 LIMS。

### 4. 指令提示說明

* **[Excel Busy]**：若您剛好在 Excel 儲存格內打字（游標閃爍中），資料會暫時無法輸入，請按下 Enter 結束編輯模式。
* **按 [r] 修改設定**：啟動程式後的 3 秒倒數期間，按下鍵盤 `r` 鍵可進入重新設定介面。
