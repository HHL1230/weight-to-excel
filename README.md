# 🚀 天平數據傳輸程式 (Weight-to-Excel)

本程式是專為Mettler Toledo天平之RS232介面設計的傳輸工具。透過即時捕捉介面資料與 **Excel 直接寫入技術** (COM API)，將天平數值填入儲存格，同時這種方式能解決傳統模擬鍵盤輸入（模擬 HID）時「輸入法干擾」及「視窗焦點被搶走」的問題。

---

## 🛠️ 安裝與準備

1. **放置路徑**：請將WeightToExcel.exe複製到您的**桌面**。
2. **環境**：
    - 確保天平已透過 RS232 線連接電腦。
    - 系統支援所有版本之 Excel。

---

## ⚙️ 首次啟動配置

第一次啟動程式時，系統會列出目前電腦偵測到的通訊埠。請根據畫面提示完成以下設定。
第2~5項為RS232通訊參數，必須與天平內設定值相同，否則無法成功傳輸數據。
[中括弧]中為預設值，若不需修改參數，直接按 **Enter** 即可：

1. **COM Port**：輸入天平連接埠的編號（例如 `COM1`）。
2. **Baud Rate (波特率)**：通訊速率，預設為 `9600`。
3. **Parity (同位檢查)**：預設為 `None`。
4. **Data Bits (資料位元)**：預設為 `8`。
5. **Stop Bits (停止位元)**：預設為 `One`。
6. **下一格移動方向**：稱量完畢後，Excel 游標要往哪裡移動？
    - `down` (下) / `up` (上)
    - `right` (右) / `left` (左)
    - `downright` (右下對角線) / `downleft` (左下對角線)
7. **閒置斷線時間**：預設為 `30` 分鐘。時間到後程式會**自動關閉視窗並釋放連線**。

以上參數會儲存在新生成的`config.json`檔案中並放在桌面，下次啟動時會自動讀取。
若程式沒找到該檔案，會重新做初始化設定。

> [!TIP]
> **如何變更設定？**
> 程式每次啟動時會有 3 秒倒數，在此期間按下鍵盤上的 **`r`** 鍵，即可進入重新設定介面。
> 或是直接修改桌面上的`config.json`檔案後重新啟動程式。

## 🌟 功能設計說明

- **背景模式**：即使 Excel 視窗被擋住（例如您正在回 Teams 或查網頁），數據依然能正確寫入。
- **輸入法避讓**：完全不受中/英文輸入法狀態影響，解決數據變成注音符號的問題。
- **指定方向**：稱重完畢後，Excel 游標會自動根據設定的方向（如右下、左下等）跳往下一格。
- **智慧斷線**：設定之閒置時間到，或同電腦由其他帳號登入時，會自動關閉程式，避免佔用COM Port。

## 📖 日常使用流程

1. **啟動程式**：雙擊桌面的 `WeightToExcel.exe`。
2. **選定儲存格**：開啟 Excel 並點選您想填入數據的位置。
3. **天平稱量**：在天平顯示數值後，按下天平上的 **Print** 鍵，數值即輸入Excel儲存格。
4. **切換至 LIMS 紀錄**：
    > [!IMPORTANT]
    > **稱重結束後，請務必手動關閉黑色的監控視窗，或等待閒置斷線時間到自動關閉。**
    > 本程式會佔用天平的通訊埠。只有在**視窗關閉後**，天平才會恢復為「純文字模式」，此時才能在 LIMS 系統中正常掃描或輸入重量。

## ⚠️ 常見問題解決 (Q&A)

**Q: 顯示 [Excel Busy]?**

- A: 如果您正在 Excel 儲存格內打字（游標閃動中），Excel 會進入編輯鎖定。請按下 **Enter** 結束輸入，天平數據才能傳入。

**Q: 程式突然消失了？**

- A: 這通常是因為達到設定之閒置時間，系統自動結束以釋放連線，這屬於正常行為，重新啟動即可。
