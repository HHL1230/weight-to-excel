Scale Monitor

Version 1.0.1 - Stable

專為實驗室設計的自動化工具。透過 RS232 直連技術，將天平重量數據精準、穩定地導入 Excel，告別手動輸入與輸入法干擾。

🚀 立即下載 ScaleMonitor.exe

支援 Windows 10/11 & Excel 全版本

⌨️

輸入法避讓技術

完全不受中/英文輸入法影響，解決傳統模擬鍵盤時數據變成注音符號或亂碼的問題。

🛡️

防連發過濾 (New)

自動過濾 OHAUS 等天平單次操作傳送的冗餘資料（如毛重、皮重），保證數據唯一性。

📊

背景寫入技術

即使 Excel 視窗不在焦點（正在處理其他工作），數據依然能準確寫入指定儲存格。

⚡

智慧自動化

支援多方向游標移動、智慧閒置斷線，完美契合 LIMS 系統作業節奏。

✨ v1.0.1 更新日誌

新增「防連發 (Debounce)」機制，支援 OHAUS AX224 等廠牌。
設定精靈加入自訂過濾時間（預設 1.0 秒）。
啟動畫面加入版本號顯示，方便確認版本。
優化與 Mettler Toledo 天平的相容性。
